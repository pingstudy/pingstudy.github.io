14. differentiate permutation from combination using venn diagram and give 2 examples each​


Answer:

Permutation:

The different ways of arranging a set of objects into a sequential order are termed Permutation.

The order is very relevant.

It denotes the arrangement of objects.

Multiple permutations can be derived from a single combination.

They can simply be defined as ordered elements.

Combination:

One of the several ways of choosing items from a large set of objects, without considering an order is termed Combination.

The order is quite irrelevant.

It does not denote the arrangement of objects.

From a single permutation, only a single combination can be derived.

They can simply be defined as unordered sets.

Step-by-step explanation:

Example

Suppose, we have to find the total number of probable samples of two out of three objects X, Y, Z. Here, first of all, you have to understand whether the problem is relevant to permutation or combination. The only means to find it is to check whether the order is necessary or not.

If the order is important, then the problem is related to permutation, and the possible number of samples will be, XY, YX, YZ, ZY, XZ, ZX. In this case, XY is distinct from the sample YX, YZ is distinct from the sample ZY and XZ is distinct from the sample ZX.

If the order is unnecessary, then the question is relevant to the combination, and the possible samples will be XY, YZ, and ZX.
