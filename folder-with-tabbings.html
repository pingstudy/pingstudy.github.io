A file is the common storage unit in a computer, and all programs and data are "written" into a file and "read" from a file. A folder holds one or more files, and a folder can be empty until it is filled. A folder can also contain other folders, and there can be many levels of folders within folders.

There are four major types of libraries:

Academic libraries serve colleges and universities.

Public libraries serve cities and towns of all types.

School libraries serve students from Kindergarten to grade 12.

To delete a file or folder (or multiple selected files), right-click on the file and select Delete. You can also select the file and hit the Delete key on the keyboard.
