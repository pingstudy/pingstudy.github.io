x + (x + 1) + (x + 2) = 314x + x + x + 1 + 2 = 3143x + 3 = 3143x = 314 - 33x = 3113x/3 = 311/3x = 311/3

To get the consecutive integers:

x = 311/3x + 1 = 311/3 + 3/3 = 314/3 (the answer)x + 2 = 311/3 + 6/3 = 317/3
