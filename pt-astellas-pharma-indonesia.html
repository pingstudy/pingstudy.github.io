An FDA officer claims that Pharma XYZ's new caplet drug contains less than 300mg of paracetamol. Test the claim using 99% confidence level. 13. The manufacturer of cigarette claims that the average nicotine content per stick is 2.1 mg. Test the claim using 90% confidence level. 14. A real estate agent claims that 60% of all condominium units built today are studio-type. Test the claim using 98% confidence level.


Answer:

Null Hypothesis: The average time to earn a bachelor's degree is 5 years. Alternative Hypothesis: The average time to earn a bachelor's degree is not 5 years.

Symbolically:

H0: µ = 5

Ha: µ ≠ 5

The probability of committing a Type 1 Error is 0.05.

If H0 is rejected, it means there is evidence to suggest that the average time to earn a bachelor's degree is not 5 years.

If H0 is not rejected, it means there is not enough evidence to suggest that the average time to earn a bachelor's degree is different from 5 years at the 95% confidence level.

Null Hypothesis: The new caplet drug contains 300mg or more of paracetamol. Alternative Hypothesis: The new caplet drug contains less than 300mg of paracetamol.

Symbolically:

H0: µ ≥ 300

Ha: µ < 300

The probability of committing a Type 1 Error is 0.01.

If H0 is rejected, it means there is evidence to suggest that the new caplet drug contains less than 300mg of paracetamol.

If H0 is not rejected, it means there is not enough evidence to suggest that the new caplet drug contains less than 300mg of paracetamol at the 99% confidence level.

Null Hypothesis: The average nicotine content per stick is 2.1 mg. Alternative Hypothesis: The average nicotine content per stick is not 2.1 mg.

Symbolically:

H0: µ = 2.1

Ha: µ ≠ 2.1

The probability of committing a Type 1 Error is 0.1.

If H0 is rejected, it means there is evidence to suggest that the average nicotine content per stick is not 2.1 mg.

If H0 is not rejected, it means there is not enough evidence to suggest that the average nicotine content per stick is different from 2.1 mg at the 90% confidence level.

Null Hypothesis: 60% of all condominium units built today are not studio-type. Alternative Hypothesis: 60% of all condominium units built today are studio-type.

Symbolically:

H0: p ≤ 0.6

Ha: p > 0.6

The probability of committing a Type 1 Error is 0.02.

If H0 is rejected, it means there is evidence to suggest that more than 60% of all condominium units built today are studio-type.

If H0 is not rejected, it means there is not enough evidence to suggest that 60% of all condominium units built today are studio-type at the 98% confidence level.
