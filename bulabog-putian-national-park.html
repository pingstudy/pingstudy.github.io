18. Bugtong Bugtong Lahat ako'y minamahal Mang-aawit ang aking tatang Ang suot ko ay putian Ang puso ko ay dilaw


Answer:

ITLOG NG MANOK

Explanation:

Lahat, ako’y minamahal

mang-aawit ang aking tatang

suot ko nama’y putian

ang puso ko’y dilaw.

Sagot: ITLOG NG MANOK

Lahat, ako’y minamahal  - Inaalagaan nang mabuti ang mga itlog ng manok dahil ang itlog ng manok ay mabuting pagkukunan ng protina at nakakatulong sa pagbaba ng timbang. Ang itlog ng manok rin ay ginagamit bilang sangkap sa iba't-ibang uri ng pagkain. Gayunpaman, pampaswerte ang itlog ng manok sa iilang tao.

Mang-aawit ang aking tatang  - Ang tatang ay nangangahulugang ama, kaya ang ama ng itlog ng manok ay ang tandang. Ang tandang ay isang mang-aawit dahil ito ay kilalang nag-titilaok sa pagsapit ng umaga.

Ang suot ko ay putian - Ang tinutukoy na suot ay ang balat ng itlog. Ito ay putian dahil ang kulay ng balat ng itlog ay puti.

Ang puso ko ay dilaw - Ang tinutukoy na pusong dilaw dito ay ang pula ng itlog (eggyolk)
