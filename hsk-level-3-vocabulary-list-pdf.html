16. List down the step by step produce in savings a file under a new file name and file type PDF file


Answer:

Here are the step-by-step instructions to save a file under a new file name and file type PDF:

Open the file you want to save as a PDF.

Click on the "File" menu located in the top-left corner of the screen.

Click on "Save As" from the dropdown menu. This will open the "Save As" dialog box.

In the "Save As" dialog box, navigate to the location where you want to save the PDF file.

Type in the new file name you want to use for the PDF file in the "File name" field.

Click on the dropdown menu next to the "Save as type" field and select "PDF (*.pdf)" as the file type.

Click on the "Save" button to save the file as a PDF.

Your file will now be saved with a new name and in the PDF file format.
