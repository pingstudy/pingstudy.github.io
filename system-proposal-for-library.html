10. Enumerate the 2 Library Classification Systems.​


Answer:

2 Main Library Classification Systems:The Dewey Decimal System

➡️ first classifies books into nine broad categories, each identified by a number in the hundreds.

The Library of Congress System

➡️ has 21 broad categories, each represented by one letter of the alphabet.

Explanation:

The Library of Congress system is an outline, much like one you would use to organize an essay. Subtopics are dealt with by indentation. The Dewey Decimal system is a hierarchy: Subtopics are dealt with by using hundreds, then tens, then units and then decimal places.

The Dewey Decimal System is used by more libraries, but the Library of Congress system is used by some of the biggest libraries -- most obviously the Library of Congress itself. Many universities use the Library of Congress system, but public libraries mostly use the Dewey Decimal System.

27. What is your opinion of having a computerized library system?


My opinion on having a computerized library system:

Computerized library system is a type of system which caters and performs libraries operations such as :  

borrowing and returning of books, adding and updating of users and information, adding and updating books and information, inventory of books and other library materials, browsing of library materials list to make searching more easier.

A computerized system means, system is running using a local network and it is not online based. Therefore, computers are connected with each other by means of cables. Each user can use the computerized system based on their levels of usage. For example, a student can borrow and return books but cannot add users and books, only the librarian can add and edit books information. Guests cannot make any transaction, but they can view library materials and books.

For me, computerized system is helpful. Librarians can get track of the books easily. They would know if anything is missing on the shelves or if any student failed to return books on time. It would also make their task easier especially when searching books and records. This would benefit not only the librarian and the school but also students and faculty members.

28. What are the parts and information found in computerized library System?


Answer:

1. Library Catalog: This is a database of all the items in the library's collection, including books, e-books, DVDs, CDs, and other physical items.

2. Electronic Resources: This includes online databases, digital databases, and other digital resources such as e-books, e-journals, and streaming media.

3. Online Reservation System: This is a system that allows users to reserve items in the library's collection, either online or in person.

4. Library Management System: This system enables the library staff to manage the library's collection, circulation, and user accounts.

5. Circulation System: This system allows users to check out and return items in the library's collection.

6. User Accounts: This includes the user's personal information, library card, and other information related to the user's account.

7. Search Engine: This is a search tool that allows users to search for items in the library's collection.

8. Reporting Tools: This includes tools for generating reports about the library's collection and users.
