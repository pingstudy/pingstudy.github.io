1. How to install video card device drivers in a windows 7​


Answer:

1. Open the Windows Start menu and go to the Control Panel.

2. Click on System and Security and Windows Update.

3. Check for Updates

Click Check for Updates if you don’t see any results yet — that tells the operating system to check for regular Windows Updates and also looks for new drivers.

4. Look for graphics card drivers that mention manufacturer names related to graphics components, such as Intel Graphics, NVIDIA, or AMD Graphics. You may see these in the main view or under the Optional Updates category. If you see one of these graphics card drivers, click on the Install button and wait for the installation to complete.
