20. what is 2/3 - 211 250=​


Answer:

-211249 2/3

Step-by-step explanation:

The asked in this question is very straightforward: Find the difference of the two given numbers.

To answer this question, rewrite the equation first:

Let n be the unknown number.

n = 2/3 - 211250

n = 2/3 + (-211250)

Now, the operation is addition. But, don't be confused! Still subtract the two numbers to find the sum.

211250 - 2/3 = 211249 2/3

We made the operation addition so that we can use the method in the next paragraph.

To know if the sign must be positive or negative, select which number has the higher absolute value. Since 211250 has a greater absolute value than 2/3, its sign will be the sign of the sum. In this case, it will be negative.

Therefore, the answer to this question is -211249 2/3 or approximately -211249.67.
