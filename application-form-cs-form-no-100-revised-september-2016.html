4. Find the approximate time of a loan made on September 15 and due on December 15, 2016.​


Answer:

3 months

Step-by-step explanation:

The approximate time of a loan made on September 15, 2016 and due on December 15, 2016 would be approximately 3 months. September 15 to October 15 is approximately 1 month, October 15 to November 15 is approximately 1 month, and November 15 to December 15 is approximately 1 month, for a total of approximately 3 months.

Answer:

The time of a loan made on September 15 and due on December 15, 2016 is approximately 90 days. To calculate this, you can count the number of days between the two dates, including the start date and excluding the end date. September has 30 days, October has 31 days, and November has 30 days, for a total of 91 days. Since the end date (December 15) is not included, the total time of the loan is approximately 90 days.

16. Set Builder form C=April,June,september and november}


Answer:

C = { April, June, September and November }

C = { x : x is a month with 30 days }

Step-by-step explanation:

Set Builder Form or Rule Method

Set builder form or rule method represents a set, with a property, such that all elements in the set satisfy that property.

The list of the elements in a set is called as roster.

How do we express set builder form or rule method?

In this method we write a general element x and ‘:’ meaning such that and state its property within brackets.

Examples:

Roster: x = { a, e, i, o, u }

Set builder form: { x: x is a vowel }

A = { 2, 4, 6, 8 }

A = { x : x is an even natural number, x ≤ 8 }
