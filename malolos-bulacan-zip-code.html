9. Itinatag ang Konstitusyon ng Malolos sa Malolos, Bulacan.


Answer:

Pagkakatatag sa Kongreso ng Malolos

Ang Kongreso ng Malolos, na kilala rin sa tawag na Kongresong Rebolusyonaryo, ay unang itinatag sa loob ng Simbahan ng Barasoain sa Malolos, Bulacan noong Setyembre 15, 1898.

Explanation:

Ang Kongreso ng Malolos, ay ang lehislatibong sangay ng pamahalaan noong isinailalim sa isang Pamahalaang Rebolusyonaryo ang Pilipinas. Ang mga miyembro nito ay pinili sa isang eleksyon na naganap din sa Malolos noong Hunyo 23 hanggang Setyembre 10, 1898. Ang mga miyembrong kasapi nito ay pawing mga iniluklok at pinili ng pangulo ng Pilipinas na kakatawan sa mga rehiyon at lalawigan ng bansa. Si Felipe G. Calderon ang namuno sa pagkakatatag ng unang kongreso ng Pilipinas. Binansagan siya bilang “Ama ng Konstitusyon ng Malolos”. Si Emilio Aguinaldo, na unang pangulo ng Pilipinas, ang namuno sa unang pagpupulong ng kongreso.

13. what is the meaning of zip code


Answer:

Meaning of Zip Code

ZIP is an acronym for Zone Improvement Plan. ZIP Code is a postal code used by the United States Postal Service. It is a group of four or nine numbers that are added to a postal address to assist the sorting of mail. People are encouraged to use the ZIP Code on letters and parcels so the mail travels more efficiently and quickly when senders use the code in the postal address.

Facts about ZIP Code

The United States Post Office Department implemented postal zones for many large cities in 1943 but ZIP Codes were introduced nationwide on July 1, 1963.

Robert Moon is considered the father of the ZIP Code.

In 1983, the U.S. Postal Service introduced an expanded ZIP Code system that it called ZIP+4. A ZIP+4 Code uses the basic five-digit code plus four additional digits to identify a geographic segment within the five-digit delivery area.
