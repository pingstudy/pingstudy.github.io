8. If the seating arrangement is to be circular,how many seating arrangements are possible?​


Answer: It's (n-1)!

Explanation: For example we have 4 persons to sit in a circular table, in order to change the arrangements, you have to keep one of them in a single position, therefore you are to change the positions of the other three. That's why it is (n-1)!

If n=4 then the possible number of arrangements is (4-1)! which is equivalent to 3! or 6.

22. If the seating arrangement is to be circular how many seating arrangements are possible?


Answer:

(see explanation)

Step-by-step explanation:

Since we are dealing with things where order matters, we use permutations. The number of arrangements of n objects in a line is given by.

[tex]n![/tex]

where ! means "factorial". Factorial notation is important when it comes to counting techniques. The factorial of a number, x, is x! (read as x factorial) given by the value of:

[tex]x! = (x)*(x-1)*(x-2)...(3)*(2)*(1)[/tex]

This means that a factorial is a product. It is a product of the variable, that number decreased by one, and so on, until that difference is 1.

Note that 0! = 1, and  1! = 1

For example, we can evaluate what 4! is

[tex]4! = 4*3*2*1= 24[/tex]

If we are dealing with a circular arrangement of objects, then we are fixing one position and the others will form a circle around that position. The formula then becomes:

[tex](n-1)![/tex]

This is called the formula for circular permutation. The number of ways n things can be arranged on a circle is:

[tex](n-1)![/tex]
