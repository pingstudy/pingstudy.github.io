2 Dimensional Fine Art Techniques

Fine art also has several techniques that we need to know, in fine art there are 4 techniques, namely:

1. Plaque Technique: The plaque technique is a painting technique using oil paint, poster paint or acrelic paint, with thick strokes to get a dense and dense result.

2. Transparent Technique: Transparent technique is a technique for painting fine art drawing using liquid paint. The strokes of the color for painting should be light so that the result is also transparent.

3. Collage Technique: Collage technique is a technique that will give a realist or abstract painting result because it is made from pieces of paper that are attached using glue.

4. 3 M Technique (Gluing, Cutting, Sticking): is a technique from fine art which is also a process of manipulating a sheet of paper which will become a 3-dimensional shape.

5. Linear technique: Linear technique is a technique for drawing objects using a line pattern from a pencil or pen.

6. Block Technique: Block technique is a technique used to cover painting objects using one color.

7. Shading Technique: Shading technique is a technique used to cover a painting object with parallel lines or cross lines using a pencil or pen.

8. Dussel Technique: The dussel technique is a technique that is used to darken and light a painting object with oblique strokes using a pencil.

9. Pointilis Technique: Pointilis technique is a technique used to blacken a painted object with several existing dots.

10. Aquarel Technique: Aquarel technique is a technique used to cover a painting object which is done by sweeping the liquid paint lightly.

11. Mosaic Techniques: Mosaic techniques are techniques used to paint by gluing 3-dimensional objects

12. Weaving Techniques: Weaving techniques are techniques that are used to overlap and also to cross the materials you want to produce woven works of art.
