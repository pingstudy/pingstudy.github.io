Simplifying Polynomials

When simplifying Polynomials, we just need to arrange the terms in descending power then add or subtract similar terms.

[tex]\\[/tex]

Binomial - a polynomial with 2 terms

1. 25a + 7a = 32a

2. -28x² + 16x² = 16x² - 28x² = -12x²

3. -76a + 128a = 128a - 76a = 52a

[tex]\\[/tex]

Multinomial - a polynomial with 3 or more terms

4. 12a² + 13a - 5a + 6a²

= 12a² + 6a² + 13a - 5a

= 18a² + 8a

5. -17ax³ + 17ax³ - 12a² + 10a²

= 17ax³ - 17ax³ + 10a² - 12a²

= -2a²
