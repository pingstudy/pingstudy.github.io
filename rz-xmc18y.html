1. What is a benefit of the rz-ami method of encoding?


Answer:

The advantage of this coding scheme is that if a long string of ones occurs, there will be no loss of synchronization. If synchronization is lost, it is easy to resynchronize at the transition. In the bipolar AMI encoding scheme, 0 is represented by no signal and 1 by either positive or negative voltage.

3. given: HZ=4a-5 RZ=3a+5 find HZ​


Answer:

Step-by-step explanation:

HZ = 4a - 5

RZ = 3a + 5

1st: Make an Equation (to find the value of a)

    HZ = RZ

4a - 5 = 3a + 5

2nd: Transpose

4a - 3a = 5 + 5

         a = 10

3rd: Substitute

HZ = 4a - 5

HZ = 4(10) - 5

HZ = 40 - 5

HZ = 35

CHECKING

d ( distance ) = 1250 m

s ( speed ) = 340.9 m/s

t ( time ) = ? unknown

The formula for Speed is speed = distance / time ( s = d / t ), we can the derivation formula which is ( t = d / s ) in order to solve the problem:

t = d / s

= 1250 m / 340.9 m/s

= 3.67 s
