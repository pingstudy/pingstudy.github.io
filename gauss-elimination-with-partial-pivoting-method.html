18. y−4x=−3 3y+2x=5 ​ substitution method and Elimination method ​


Answer:

Step-by-step explanation:

To solve the system of equations:

y - 4x = -3

3y + 2x = 5

using the substitution and elimination methods, we can follow these steps:

Substitution Method:

Solve one of the equations for either x or y.

For example, we can solve the first equation for y:

y = 4x - 3

Substitute the expression obtained in step 1 into the other equation for the same variable.

We substitute y = 4x - 3 into the second equation:

3(4x - 3) + 2x = 5

Solve for the remaining variable.

12x - 9 + 2x = 5

14x = 14

x = 1

Substitute the value obtained in step 3 into either equation to solve for the other variable.

We substitute x = 1 into the first equation:

y - 4(1) = -3

y - 4 = -3

y = 1

Therefore, the solution to the system of equations is (x, y) = (1, 1).

Elimination Method:

Multiply one or both equations by constants to obtain opposite coefficients for one of the variables.

We can multiply the first equation by 3 and the second equation by -4 to obtain opposite coefficients for x:

3y - 12x = -9

-12x - 8x = -20

Add the resulting equations to eliminate one of the variables.

3y - 12x + (-20) = -9

3y - 12x - 20 = -9

3y - 12x = 11

Solve for the remaining variable.

3y - 12(1) = 11

3y - 12 = 11

3y = 23

y = 23/3

Substitute the value obtained in step 3 into either equation to solve for the other variable.

We substitute y = 23/3 into the first equation:

(23/3) - 4x = -3

23 - 12x = -9

12x = 32

x = 8/3

Therefore, the solution to the system of equations is (x, y) = (8/3, 23/3).

20. 2x + 2y =9 and 2x + 3y = -2 substitution method and Elimination method​


Step-by-step explanation:

Substitution Method:

We can solve the system of equations:

2x + 2y = 9 (equation 1)

2x + 3y = -2 (equation 2)

From equation 1, we can isolate x:

2x = 9 - 2y

x = (9 - 2y)/2

We can substitute this expression for x in equation 2:

2[(9 - 2y)/2] + 3y = -2

Simplifying and solving for y:

9 - 2y + 3y = -2

y = -11

Now we can substitute this value of y back into equation 1 to find x:

2x + 2(-11) = 9

2x = 31

x = 31/2

Therefore, the solution to the system of equations is (x, y) = (31/2, -11).

Elimination Method:

We can solve the system of equations:

2x + 2y = 9 (equation 1)

2x + 3y = -2 (equation 2)

We can multiply equation 1 by -3 and equation 2 by 2 to eliminate x:

-6x - 6y = -27

4x + 6y = -4

Adding the two equations together:

-2x = -31

Dividing by -2:

x = 31/2

Now we can substitute this value of x back into equation 1 to find y:

2(31/2) + 2y = 9

31 + 2y = 9

y = -11

Therefore, the solution to the system of equations is (x, y) = (31/2, -11).
