4. How many bcd code bits and how many straight binary bits would be required to represent the decimal number 643


Answer:12 BCD-coded bits, 10 straight binary bits

Explanation:

[tex]643_{10}[/tex] can be converted to BCD by following the conversion equivalent below:

Decimal = BCD

0 = 000

1 = 0001

2 = 0010

3 = 0011

4 = 0100

5 = 0101

6 = 0110

7 = 0111

8 = 1000

9 = 1001

From the conversion equivalent above, [tex]3_{10}[/tex] is equal to 0011, [tex]4_{10}[/tex] is equal to 0100, and [tex]6_{10}[/tex] is equal to 0110. Now, we can combine them. [tex]643_{10} = 011001000011_{BCD}[/tex]

We need to count the BCD-coded bits. Since each number has 4 bits, we multiply it by the number of digits which is 3, to get the total number of 12 BCD-coded bits.

[tex]643_{10}[/tex] will be converted to binary using the procedures below:

643 ÷ 2 = 321              remainder         1

321 ÷ 2 = 160               remainder         1

160 ÷ 2 = 80                remainder         0

80  ÷ 2 = 40                remainder         0

40  ÷ 2 = 20                remainder         0

20  ÷ 2 = 10                 remainder         0

10  ÷ 2 = 5                   remainder         0

5 ÷ 2 = 2                      remainder         1

2 ÷ 2 = 1                      remainder         0              

1 ÷ 2 = 0                      remainder         1

Reading it from the lower part to the top part, we get [tex]1010000011_{2}[/tex]

We must count the bits of the binary [tex]1010000011_{2}[/tex] which gives us 10 straight binary bits.
