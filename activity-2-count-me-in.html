28. Activity 2: Count Me In! 1) 2a2 + 7c = Solution: Answer: 2) a2 + 3b + 5c2 = Solution: Answer: 3) a3b2 + 6c2 = Solution: Answer:​


1.) If a = 3, b = 6, and c =4

2a² + 7c

=> 2(3)² + 7(4)

=> 2(9) + 7(4)

=> 18 + 28

=> 46

2.) If a = 3, b = 6, and c =4

a² + 3b + 5c²

=> 3² + 3(6) + 5(4)²

=> 9 + 3(6) + 5(16)

=> 9 + 18 + 80

=> 107

3.) If a = 3, b = 6, and c =4

a³b² + 6c²

=> 3³6² + 6(4)²

=> (27)(36) + 6(16)

=> 972 + 96

=> 1068
