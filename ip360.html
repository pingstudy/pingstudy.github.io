6. The discount of a stand fan during a sale is 20%. The discount's worth is P360. What is the original price?​


Answer:

To find the original price of the stand fan, we can set up the following equation using the information provided:

Discount = (Original Price x 20%) = P360

We know that the discount is 20% of the original price, so we can represent 20% as the decimal 0.20. We can then substitute that into the equation and solve for the original price:

Discount = (Original Price x 0.20) = P360

To find the original price, we can divide both sides of the equation by 0.20:

Original Price = Discount/0.20 = P360/0.20 = P1800

So the original price of the stand fan is P1800. It's confirmed by multiplying the original price by 20% which should give the discount's worth P1800*0.20 = 360.
