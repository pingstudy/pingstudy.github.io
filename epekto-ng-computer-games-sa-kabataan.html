1. Ano ang epekto ng computer games sa kabataan


Ang epekto ng computer games sa kabataan

Maraming epekto ang paglalaro ng computer games sa mga kabataan. Mayroong positibo at mayroon ding negatibong epekto. Ang computer game ay isang laro na gumagamit ng computer upang ito ay malaro. Kadalasan, nakakaaliw ang graphics at maganda ang effects ng isang computer game. Kinakailangan ring gumamit ng internet upang ito ay malaro.

Positibong epekto: Ang mga kabataan na naglalaro ng computer games ay mas naipapahayag nila ang kanilang sarili sa paglalaro. Minsan, ang isang bata ay mahiyain o hindi madalas nakikihalibulo sa kanilang kapwa. Pero sa pamamagitan ng paglalaro, sila ay nagiging madaldal at natututo makisalamuha sa kanilang kapwa kahit ito ay sa mundo lamang ng mga laro. Maraming makikilala na kapwa nila bata ang mga batang naglalaro ng computer games. Sa pamamagitan ng computer games, ang mga bata ay nakakakilala ng mas maraming bata na saklaw ang parehong interes o hilig kaya mas marami silang nakakasundo at nakakausap.

Negatibong epekto: Ang mga kabataan na naglalaro ng computer games ay nalululong sa bisyo. Kung minsan, nagiging mitsa ng pustahan o sugal ang paglalaro ng computer games. Kung sinong grupo ang mananalo, sila ang tatanggap ng pera sa pustahan samantalang kung sino ang talo, sila ang magbabayad. Ang paglalaro ng computer games ay nagiging sanhi ng paglayo ng loob ng mga kabataan sa kanilang magulang at pamilya dahil sa kanilang labis na pagkalulong sa paglalaro at mga nakakausap nilang tao sa laro, hindi na nila nabibigyan ng oras ang kanilang mga magulang at pamilya.

3. Ang masamang epekto ng pag lalaro ng computer games ng mga kabataan sa kanilang grado sa pang akademikoâ€‹


Answer:

Kung ang isang estudyante ay hindi marunong mag-manage ng kanyang oras, ang paglalaro ng computer games ay maaaring maghantong sa kanya sa pagkalulong dito. Mas uunahin pa nya ang paglalaro kesa sa pag-aaral kung kaya mas malamang e hindi sya makakasagot sa mga asignatura. At dahil diyan, mas malamang na siya ay makakuha ng mababang grado.

Ngunit kung ang estudyante ay marunong mag-manage ng kanyang oras, alam nyang limitahan ang paglalaro, alam nya ang tamang oras ng pag-aaral, maaaring ang paglalaro ng computer games ay maging inspirasyon nya.

Ating tandaan, hindi masama ang paglalaro ng computer games. Ang masama ay ang malulong sa paglalaro nito at kalimutan ang ibang responsibilidad.
