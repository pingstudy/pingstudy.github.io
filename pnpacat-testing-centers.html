2. The number of tests for a disease varies directly as the number of testing centers and inversely as the cost of the test. There were 30 000 tests done in 30 testing centers in City X. Each test costs PhP 4000. How many tests were done in City Z with 25 testing centers offering the test at PhP 4500? *​


Direct variation: y = kx

Inverse variation: y = [tex]\sf\large{ \frac{k}{x}}[/tex]

[tex]{}[/tex]

The number of tests - n

number of testing centers - c

the cost of the test - t

[tex]{}[/tex]

n = kc

n = [tex]\sf\large{ \frac{k}{t}}[/tex], so

n = [tex]\sf\large{ \frac{kc}{t}}[/tex]

[tex]{}[/tex]

30000 = [tex]\sf\large{ \frac{30k}{4000}}[/tex]

120000000 = 30k

k = 4000000

[tex]{}[/tex]

n = [tex]\sf\large{ \frac{4000000c}{t}}[/tex]

[tex]{}[/tex]

c = 25

t = 4500

[tex]{}[/tex]

n = [tex]\sf\large{ \frac{4000000 × 25}{4500}}[/tex] = 22,222.22 ≈ 22,222

[tex]{}[/tex]

Answer: 22,222
