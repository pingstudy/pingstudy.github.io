9. find the circumference of a circle if its area is 706 point 5 cm²​


Answer:

94.2cm

Step-by-step explanation:

A = π r²

706.5cm² = 3.14r²

706.5cm²÷3.14 = 3.14r²÷3.14

r² = 225cm²

√r² = √225cm

r = 15cm

C = 2π(r)

C = 2(3.14)(15cm)

C = 94.2cm
